# 公招识别

会先过一遍`dhash`，若汉明距离小于阈值，则进行OCR识别，计算公招结果

## 致谢

- 差异哈希 [dHash](https://github.com/benhoyt/dhash)
- 公招算法及数据 [AAH](https://github.com/ninthDevilHAUNSTER/ArknightsAutoHelper)
